array dan hashmap

kesimpulan:

1. sering digunakan untuk checking dengan hasil baru atau nilai boolean.
2. ada juga digunakan untuk manipulating data menjadi data yang sudah di modifikasi bedasarkan suatu kondisi.

two pointer

kesimpulan:

1. sering digunakan untuk optimisasi brute force dengan menggunakan dua pointer dari dua arah yang berbeda. (biasanya awal dan akhir)
2. sering digunakan untuk processing data baru
3. kadang digunakan untuk validasi

stack

kesimpulan
1. sering digunakan untuk validasi antara data baru dengan data lama dengan menggunakan pointer terakhir
2. sering digunakan untuk processing data baru
3. kadang digunakan untuk validasi
